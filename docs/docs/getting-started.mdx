Quick Start Guide to Getting Started with "View Exports SVG" in Your Project.

## What is View Exports SVG?

**View Exports SVG** is a Visual Studio Code extension designed to streamline the process of accessing and utilizing SVG icons.
With this extension, you can effortlessly explore and search for SVG icons within your project.

## üß† First Run

Once opened, the extension will:

- Scan your project for exported SVG components (`.jsx`, `.tsx`, `.js`, `.ts`).
- Display them in a dedicated DevTools panel.
- Let you preview icons instantly.

You can start interacting right away, or explore advanced features next.

---

## üéâ You're All Set!

Congratulations! You have successfully installed and opened `View Exports SVG`. Start exploring your SVG components directly within Visual Studio Code.
