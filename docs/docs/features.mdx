**View Exports SVG** extends Visual Studio Code with a complete toolkit for discovering, previewing, and managing SVG components in your workspace.

This page provides an overview of the key features that make it an essential tool for frontend developers and design systems.

## üß© Framework Compatibility

The extension supports modern frontend stacks by analyzing exported components in multiple frameworks.

- **React** (default support)
- **Preact**
- **TypeScript and JavaScript**

:::tip Pro tip
Even if your project mixes `.js`, `.jsx`, `.ts`, and `.tsx` files, View Exports SVG can still
detect and display all exported components seamlessly.
:::
