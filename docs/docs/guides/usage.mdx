Once **View Exports SVG** is installed, you can explore, preview, and manage your SVG components directly within Visual Studio Code.

This guide focuses on how to **open**, **navigate**, and **interact** with the extension's main features.

## üß≠ Opening the Extension

You can open **View Exports SVG** in several ways depending on your workflow.

### From the Context Menu

Right-click one or more files and select **"View Exports SVG üîç"** from the context menu to scan and display available components.

### From the Command Palette

Open the Command Palette (press <Shortcut mac={['command', 'shift']} windows={['ctrl', 'shift']}>P</Shortcut>), then type and run `View Exports SVG: Start Scanning` to scan your entire workspace.

Alternatively, run `View Exports SVG üîç` to access recent and favorite icons.

## ‚öôÔ∏è Browsing and Managing Icons

Once opened, the `View Exports SVG` panel displays all detected SVG components from the scanned files.

Each icon appears in a grid view with its preview, component name, and available actions.

<div align='center' className='mt-4'>![The main UI of the View Exports SVG panel showing a grid of SVG icons.](/screens/principal-ui.webp)</div>

### Common actions

- Click actions on an icon
  - Copy the name to the clipboard.
  - Open the DevTools ([**see next section**](/docs/guides/devtools)).
  - Add to the recent icons list (configurable option ‚Äî [**Go to settings**](/docs/guides/settings#replace-real-path)).
- Mark / unmark as favorite
  - Hover over the icon and click the star in the top-right corner to toggle.
- Context menu (right-click on an icon)
  - Copy name or code snippet.
  - Open the icon in the editor.
  - Copy the icon to the clipboard as PNG or SVG.
  - Download the icon as PNG or SVG.

## üß© Drag & Drop Support

To load and analyze files, execute from the Command Palette, run `View Exports SVG üîç`.

On the start screen you can drag and drop one or more files; the extension will read their contents, search for SVG icon components, and make them available for preview and management.

<div align='center' className='mt-4'>
  <video src='/screens/upload-files-drag-drop.webm' controls muted />
  <p className='mt-2 text-sm text-gray-600 dark:text-gray-400'>Drag & Drop files into the View Exports SVG panel to preview their SVG components.</p>
</div>

:::caution Security Limitation
Due to VS Code's WebView sandbox restrictions, direct access to file paths is blocked.\
When you drag and drop files into the panel, the extension reads their contents and creates temporary
in-memory versions for safe preview.

<p className='ml-auto'>[Read more ‚Üí](/docs/resources/troubleshooting#replace-real-path)</p>
:::
