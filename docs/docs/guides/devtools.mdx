The **DevTools panel** is the advanced workspace where you can inspect, preview, and edit SVG components in real time‚Äîwithout leaving Visual Studio Code.

It is designed to help you:
- Visually validate icons before using them
- Experiment with props and styles
- Inspect component metadata and structure

## üß† DevTools Overview

The DevTools panel is divided into two main sections:

- **Playground** ‚Äî Interactive editor and live preview
- **Component Info** ‚Äî Read-only metadata about the selected icon

Each section updates automatically when you select a different component.

---

## üß© Playground

The **Playground** lets you experiment with SVG components dynamically.
It combines a live preview, a real-time code editor, and quick-access actions.

### ‚úèÔ∏è Real-time Editing

Use the built-in editor to modify your component props.
All changes are reflected instantly in the preview.

### ‚ö° Quick Editor Actions

The editor toolbar provides quick access to common actions:

- **Show / Hide code** ‚Äî Toggle editor visibility
- **Copy code** ‚Äî Copy the current editor content
- **Reset code** ‚Äî Restore the initial state

Additional actions are available under the **More actions** menu:

- **Open in Editor**
- **Copy as SVG / PNG**
- **Download as SVG / PNG**

### üß∞ Playground Menu Actions

At the top-right of the Playground, you'll find an **actions menu** (three dots icon) for managing the environment.

The menu includes the following options:

- **Reload Playground** ‚Äî Resets the preview and editor.
- **Enable Word Wrap** ‚Äî Toggles line wrapping in the editor.
- **Reload Editor** ‚Äî Rebuilds the code editor.

## ‚ÑπÔ∏è Component Info

The **Component Info** section displays read-only metadata about the selected icon.

| Field                | Description                                      |
| -------------------- | ------------------------------------------------ |
| **Component name**   | The exported name of the icon component.         |
| **Type declaration** | Whether it's defined as a variable or function.  |
| **Is exported?**     | Indicates if the component is publicly exported. |
| **Is animated?**     | Detects motion or animation within the SVG.      |
| **Is favorite?**     | Shows whether it's marked as favorite.           |

:::note
This section is informational only and updates automatically when you select another component.
:::
</div>
