This quick start will help you install it and open the panel for the first time.

## üß† First Run

Once opened, the extension will:

- Scan your project for exported SVG components (`.jsx`, `.tsx`, `.js`, `.ts`).
- Display them in a dedicated DevTools panel.
- Let you preview icons instantly.

You can start interacting right away, or explore advanced features next.
