## ‚ú® Overview

**View Exports SVG** is a powerful Visual Studio Code extension that revolutionizes how developers interact with SVG components.
It goes far beyond simple file previews ‚Äî offering a complete workflow for **discovering**, **inspecting**, and **managing** your SVG assets without leaving your editor.

Whether you're working on a large icon library or dynamic SVG components, View Exports SVG gives you the visualization power and DevTools you need.
